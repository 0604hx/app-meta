### 本地缓存
> 编辑过程中，可以申请保存缓存到本地（默认为 indexedDB）

`Editor.vue` 监听`editor.cache`事件，将当前编辑页的内容（由具体编辑页传递）缓存起来

当前编辑页则监听`editor.cache.read`事件，拿到缓存内容

同一页面缓存数量上限为`10`。
