# APP-META 终端命令行工具
> 借助`终端工具`便捷使用平台功能

# 概述

## 功能清单

* `简介 info`：用于检测是否联通（显示服务器时间）
* `快应用 page`
    * `详情 [id]`：详细指定页面的详情（launch 数会加 1）
    * `部署 deploy`：将本地打包好的小程序或后端服务更新到平台
    * `重启 restart`：重启后端服务
* `配置 config`：修改或获取配置信息
* `设置用户 whoami`：修改或获取用户ID
